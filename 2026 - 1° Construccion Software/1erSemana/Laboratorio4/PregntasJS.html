Cuestionario de Fundamentos de JavaScript

1. Diferencias y semejanzas entre Java y JavaScript

Semejanzas:

  • Ambos usan sintaxis similar basada en C.
  • Ambos permiten programación orientada a objetos.
  • Ambos usan estructuras como if, for, while.
  • Ambos trabajan con variables y funciones.

Diferencias:

  • Java es de tipado estático; JavaScript es de tipado dinámico.
  • Java se compila; JavaScript es interpretado.
  • Java no realiza coerción automática como JavaScript.
  • Java se usa en aplicaciones empresariales; JavaScript principalmente en desarrollo web.

2. Métodos del objeto Date

  • getFullYear()
  • getMonth()
  • getDate()
  • getHours()
  • getMinutes()
  • getSeconds()
  • getTime()
  • setFullYear()

3. Métodos de los arreglos

  • push()
  • pop()
  • shift()
  • unshift()
  • slice()
  • splice()
  • map()
  • filter()
  • forEach()

Diferencia importante: slice() no modifica el arreglo original, mientras que splice() sí lo modifica.

4. Declarar variable con alcance local

Una variable local se declara dentro de una función utilizando let, const o var. 

function ejemplo() {
    let x = 10;
}

La variable solo existe dentro de esa función.

5. Implicaciones de usar variables globales

  • Posibles colisiones de nombres.
  • Efectos secundarios inesperados.
  • Dificultad para depurar errores.
  • Menor modularidad del código.

Se recomienda usar variables locales para evitar estos problemas.
