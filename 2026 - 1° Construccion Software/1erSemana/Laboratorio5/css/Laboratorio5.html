Aplicación con Bulma

Laboratorio 5: Frameworks de estilo, CSS y JS

Pregunta 1

¿Es responsivo mi diseño?
Sí, Bulma es un framework basado en Flexbox, lo que permite que el diseño se adapte automáticamente a diferentes tamaños de pantalla.

Pregunta 2

¿Qué componentes utilicé del framework?
Utilicé Hero para el encabezado principal, Card para organizar la información y Container para centrar el contenido.

Pregunta 3

¿Qué es Material Design?
Material Design es un sistema de diseño creado por Google que establece principios visuales y de interacción para crear interfaces modernas e intuitivas.

Pregunta 4

¿Cuáles son sus principales características?

  • Uso de sombras y capas para simular profundidad
  • Animaciones suaves y coherentes
  • Colores definidos y jerarquía visual
  • Diseño responsivo y adaptable

Pregunta 5

¿Qué ventajas tiene usar Bulma frente a CSS tradicional?
Reduce el tiempo de desarrollo y mejora la organización del diseño mediante clases predefinidas.
