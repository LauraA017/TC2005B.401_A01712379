Preguntas y Respuestas sobre CSS

1. Como ingeniero de software ¿cuál es tu recomendación sobre el uso de !important en un CSS?

Mi recomendación es evitar el uso excesivo de !important. Solo debería utilizarse en casos muy específicos, ya que rompe la jerarquía natural del CSS y puede dificultar el mantenimiento del código. Es mejor organizar correctamente los selectores y la especificidad.

2. Si se pone una imagen de fondo en una página HTML, ¿por qué debe escogerse con cuidado?

La imagen de fondo debe escogerse con cuidado porque puede afectar la legibilidad del texto y el rendimiento del sitio. Si la imagen es muy pesada, hará que la página cargue más lento. Además, si tiene muchos colores o contraste fuerte, puede dificultar la lectura.

3. Como ingeniero de software, ¿cuál es tu recomendación al elegir las unidades de un propiedad de estilo entre %, px y pt?

Recomiendo usar % para diseños responsivos, ya que se adapta al tamaño de la pantalla. Los px son útiles para medidas exactas en pantalla. Los pt se utilizan más en impresión y no tanto en diseño web. Es importante elegir la unidad según el contexto.

4. ¿Por qué el uso de una versión minimizada del CSS mejora el rendimiento del sitio?

Una versión minimizada elimina espacios, saltos de línea y comentarios innecesarios, lo que reduce el tamaño del archivo. Esto permite que el navegador descargue el CSS más rápido, mejorando el tiempo de carga del sitio.
